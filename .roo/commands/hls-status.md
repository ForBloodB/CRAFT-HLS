# /hls-status

Show current HLS contract/run status.

Read these files when present:

- `contract_meta.json`
- `diagnosis.json`
- `workflow_status.json`
- `workflow_events.jsonl`
- `contract_token_report.json`
- `workflow_token_summary.json`
- latest run `result.json`
- latest run `token_report.json`
- `resolution_report.json`

Summarize current stage, token usage by stage, budget usage, and next required
action.
